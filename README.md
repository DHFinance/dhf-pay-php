# dhf-pay-php
PHP SDK to integrate with DHFinance in minutes.
# Install
 Add repositorie to composer.json
```sh
"repositories": [
        {
            "type": "git",
            "url": "https://github.com/DHFinance/dhf-pay-php.git"
        }
    ]
    
```

and add sdk to composer
```sh
 "require": {
        dhfinance/dhf-pay-php": "*"

    }
```
